# Warp Development Guide

This repository is designed for Warp AI development workflows.

Project setup process:

1. Copy /docs-template → /docs
2. Copy /skills-template → /skills
3. Complete product documentation
4. Generate a build plan
5. Begin implementation

Typical implementation order:

1. Data model
2. Backend services
3. Core UI screens
4. Integrations
5. Analytics
6. QA
