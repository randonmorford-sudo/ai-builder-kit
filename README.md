# AI Builder Kit

Reusable templates for building AI-assisted applications with Warp and GPT.

## Contents

### docs-template
Product documentation templates used to define projects before building.

Includes:
- project overview
- user problem
- success metrics
- product requirements
- MVP scope
- architecture
- launch planning
- analytics
- QA

### skills-template
Reusable engineering guidance for AI coding agents.

Includes:
- Next.js architecture
- Supabase backend patterns
- TypeScript standards
- UI component patterns
- testing and quality
- AI coding guidelines

## Usage

When starting a new project:

1. Copy `/docs-template` → `/docs`
2. Copy `/skills-template` → `/skills`
3. Fill out product docs
4. Begin development with Warp
