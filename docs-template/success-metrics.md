\# Success Metrics

These metrics determine whether the product is successful.

\---

\#\# Primary Metric

The most important measure of success.

Examples:

daily active users    
projects created    
notes written    
subscriptions purchased

\---

\#\# Secondary Metrics

Other useful indicators.

Examples:

user retention    
conversion rate    
session length

\---

\#\# 30-Day Success Criteria

Define realistic goals for the first 30 days after launch.
