# AI Coding Guidelines Skill

These rules guide AI agents modifying this repository.

## Read Project Context

Before implementing features, review:

/docs
/skills

These directories define product scope and engineering standards.

---

## Follow the Build Plan

Implementation should follow the phases defined in:

docs/build-plan.md

Do not implement features outside the current phase.

---

## Preserve Architecture

Do not introduce new frameworks or major dependencies without justification.

Follow existing folder structures.

---

## Incremental Changes

Make small, focused changes.

Avoid rewriting large sections of code unless necessary.

---

## Output Expectations

When modifying code:

- explain changes
- list modified files
- highlight potential side effects
